# ❤️ Heart Disease Prediction App

An AI-powered Heart Disease Prediction Web Application built using Machine Learning and Streamlit.

This application predicts the probability of heart disease based on patient health information and provides an interactive dashboard with risk analysis.

# 🧠 Machine Learning Model

The application uses:

* Logistic Regression
* Feature Scaling using StandardScaler
* Scikit-learn

---

# 📊 Input Features

The model predicts heart disease using:

* Age
* Gender
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol
* Fasting Blood Sugar
* Resting ECG
* Maximum Heart Rate
* Exercise Induced Angina
* OldPeak
* ST Slope

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/heart-disease-predictor.git
```

## Open Project Folder

```bash
cd heart-disease-predictor
```

## Install Requirements

```bash
pip install -r requirements.txt
```

## Run Streamlit App

```bash
streamlit run app.py
```
